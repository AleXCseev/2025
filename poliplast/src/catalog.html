Оздоблення інтер’єрів та фасадів плиткою – одна з найпопулярніших технологій у будівництві. Це не могло пройти повз увагу великого українського виробника будівельних матеріалів — компанії Поліпласт. Купити клей для плитки під цією маркою можна для різних видів робіт:

  • у приміщеннях звичайної вологості;
  • для виробничих приміщень;
  • у ванних та кухнях;
  • для фасадів.

Клей для плитки купити в Києві можна для різних видів основ – цегли, штукатурки, газобетону, гіпсокартонного листа. Всі склади розроблені з урахуванням специфіки конкретного матеріалу як стінового, так і виду плитки.
