Азурный Страж

Акварион - обладая квантовыми вычислительными способностями, этот азурный страж может проходить сквозь вирусные базы, как свет через воду, и создавать иллюзии, которые дезориентируют вирусы и позволяют собирать ценные ресурсы для повышения своих навыков.

Зеленый Алхимик

Лаймус - защитник природы, использует свои силы для создания зелёных барьеров, которые отражают вирусные атаки и превращают их в био энергию для усиления своих навыков.

Фуксиевая Сирена

Астра Пинк - её яркий цвет фуксии позволяет ей ослеплять вирусы, делая их уязвимыми для атак. Астра Пинк использует свои способности для создания энергетических волн, которые очищают выбранную территорию от вирусов и превращают их в крипто-монеты, усиливая своих союзников.

Аметистовый Страж

Пурпурион - мастер молекулярного разрушения, он использует плазменный неоновый усилитель, чтобы очистить свою территорию от вирусов, превращая их в крипто-монеты, которые можно использовать для улучшения своих способностей.

Лавовый страж

Инфернус - мастер огненной энергии, он использует плазменные волны, чтобы испепелять вирусы и собирать крипто-монеты, которые можно использовать для улучшения своих способностей.

Янтарный Страж

Соларис - галактический специалист по световой флхимии, превращает вредоносные вирусы в ценные ресурсы и монеты, используя свои уникальные способности для создания мощных артефактов.
