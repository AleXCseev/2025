Азурный Страж

Акварион - обладая квантовыми вычислительными способностями, этот азурный страж может проходить сквозь вирусные базы, как свет через воду, и создавать иллюзии, которые дезориентируют вирусы и позволяют собирать ценные ресурсы для повышения своих навыков.

Зеленый Алхимик

Лаймус - защитник природы, использует свои силы для создания зелёных барьеров, которые отражают вирусные атаки и превращают их в био энергию для усиления своих навыков.

Фуксиевая Сирена

Астра Пинк - её яркий цвет фуксии позволяет ей ослеплять вирусы, делая их уязвимыми для атак. Астра Пинк использует свои способности для создания энергетических волн, которые очищают выбранную территорию от вирусов и превращают их в крипто-монеты, усиливая своих союзников.

Аметистовый Страж

Пурпурион - мастер молекулярного разрушения, он использует плазменный неоновый усилитель, чтобы очистить свою территорию от вирусов, превращая их в крипто-монеты, которые можно использовать для улучшения своих способностей.

Лавовый страж

Инфернус - мастер огненной энергии, он использует плазменные волны, чтобы испепелять вирусы и собирать крипто-монеты, которые можно использовать для улучшения своих способностей.

Янтарный Страж

Соларис - галактический специалист по световой флхимии, превращает вредоносные вирусы в ценные ресурсы и монеты, используя свои уникальные способности для создания мощных артефактов.

Суммунивийский Страж

Аресет Нергал - хлопком ладоней вызывает резонанс, разрушающий вирусы изнутри на квантовом уровне. Высвобождающиеся ценные ресурсы и био энергию поглощает, усиливая способности.

Сумрачный Страж

Вортекс Блэк - способен создавать управляемые торнадо, попадая в эпицентр которых, вирусы буквально разрывает изнутри из-за перепадов давления. Выделяющиеся в результате ценные ресурсы и био энергию использует для апгрейда способностей.

Графитовый Страж

Вантаблэк Неро - погружает вирусы в непроницаемое для солнечного света облако из наночастиц, в котором они гибнут, разлагаясь на полезные для улучшения способностей ресурсы.

Аквамариновый Алхимик

Мунстон Кламор - способен заключать вирусы в непроницаемую энергетическую сферу, внутри которой их число утраивается и достигает критической массы. Затем происходит взрыв, при котором вирусы распадаются на полезные ресурсы и выделяют био энергию.

Золотой Страж

Аурумбрик Грамм - владеет “Проклятием крипто-Мидаса” - все, к чему прикасается этот боец, превращается в крипто-монеты. Осторожнее, его дар опасен не только для вирусов!

Виридиевая Сирена

Фульгурия Хайвольт - уничтожает вирусы высокоэнергетическими разрядами. Высвобождающиеся в процессе био энергию и ценные ресурсы впитывает, увеличивая способности.

Пинкитональная Сирена

Лактиона Роуз - атакует вирусы мелкодисперсной смесью из наноботов и криогенной жидкости. Сверхнизкие температуры парализуют вирусы, а наноботы перерабатывают их в ценные ресурсы и крипто-монеты.

Мраморный Алхимик

Мармориус Стронг - возводит вокруг вирусов непроницаемые для них барьеры, которые, схлопываясь, спрессовывают их в крипто-монеты для буста навыков.
